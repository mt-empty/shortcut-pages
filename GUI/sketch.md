    `O                             {{Oval}}

    `⌘ U                           {{Underline}} 
    `Alt ⌘ +                       {{Increase Font Size}} 
    `Alt ⌘ –                       {{Decrease Font Size}} 
    `Alt Control L                 {{Increase Character Spacing}} 
    `Alt Control T                 {{Decrease Character Spacing}}

    `Shift ⌘ O                     {{Convert Text to Outlines}} 
    `⌘ Shift {                     {{Align Left}} 
    `⌘ Shift |                     {{Align Center}} 
    `⌘ Shift }                     {{Align Right}} 
    `Control ⌘ Space               {{Special Characters}}
